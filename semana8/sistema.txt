Comando: uname -a

Este comando muestra información completa del sistema operativo, incluyendo el nombre del sistema, versión del kernel, arquitectura del procesador y fecha de compilación.

Comando: whoami

Este comando muestra el nombre del usuario que actualmente está utilizando el sistema.

Importancia:

Conocer la información del sistema es importante para administrar servidores porque permite identificar el entorno de trabajo, verificar compatibilidad de software, realizar diagnósticos, aplicar configuraciones adecuadas y gestionar correctamente los permisos de usuario.
